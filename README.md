This repository stores the study materials and assignments of Thi Ngoc Phung Cao, an exchange student from Ho Chi Minh City University of Technology and Education (HCMUTE), Vietnam, currently studying at Cracow University of Technology (Politechnika Krakowska), Poland.

The repository is organized into separate branches, each corresponding to a specific course taken during the exchange program:
- Introduction_to_Database
- Introduction_MachineLearning
- Elements_of_Artificial_Intelligence
- DataMining_with_R

Each branch contains lecture notes, lab exercises, reports, and project files related to the respective course.
